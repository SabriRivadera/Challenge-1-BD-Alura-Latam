# 📊 Análisis de Ventas de Tiendas Alura

Proyecto realizado como parte del Challenge 1 de Alura Latam - Ciencia de Datos. En este desafío se analiza el rendimiento de cuatro tiendas de la cadena Alura Store para identificar cuál presenta menor eficiencia y así recomendar cuál debería venderse.

## 📌 Descripción

### 1. Analizar datos de las tiendas
Evaluación de distintos indicadores de rendimiento, incluyendo:
- Ingresos totales por tienda
- Ventas por categoría de producto
- Calificación promedio de clientes
- Productos más vendidos y menos vendidos
- Costo de envío promedio

### 2. Crear visualizaciones
Se utilizan gráficos para comparar el desempeño de las cuatro tiendas:
- Gráficos circulares
- Gráficos de barras
- Comparaciones entre tiendas
- Visualización de productos más y menos vendidos

### 3. Proporcionar una recomendación final
Con base en el análisis, se redacta una conclusión que permita responder:
- ¿Qué tienda debería venderse?
- ¿Qué métricas respaldan esa decisión?
